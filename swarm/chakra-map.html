7. Sahasrara (Crown)
Top of head
Anatomical Correlate
  • Cerebral cortex (higher consciousness, integration)
  • Pineal gland (melatonin, circadian rhythm, DMT synthesis)
Physiological Function
Sleep-wake cycles, consciousness states, integration of all sensory/cognitive input
Neurotransmitters
Serotonin, melatonin, potentially endogenous tryptamines
Energetic Function
Connection to universal consciousness, transcendence of ego, unity experience
6. Ajna (Third Eye)
Between eyebrows
Anatomical Correlate
  • Pituitary gland (master endocrine regulator)
  • Cavernous plexus
  • Prefrontal cortex (executive function, intuition)
Physiological Function
Hormonal regulation, vision processing, pattern recognition, decision-making
Hormones
Growth hormone, prolactin, ACTH, TSH, FSH, LH (controls other glands)
Energetic Function
Intuition, insight, clairvoyance, pattern recognition beyond logic
5. Vishuddha (Throat)
Throat
Anatomical Correlate
  • Thyroid gland (metabolism regulation)
  • Parathyroid (calcium/bone health)
  • Cervical plexus
  • Larynx, vocal cords
Physiological Function
Metabolic rate, speech production, calcium regulation, temperature control
Hormones
T3, T4 (thyroid hormones), calcitonin, PTH (parathyroid hormone)
Energetic Function
Expression, communication, speaking truth, creative articulation
4. Anahata (Heart)
Center of chest
Anatomical Correlate
  • Cardiac plexus
  • Thymus gland (immune system, T-cell maturation)
  • Heart, lungs
  • Vagus nerve (parasympathetic)
Physiological Function
Circulation, oxygenation, immune response, heart rate variability, stress regulation
Neurotransmitters/Hormones
Oxytocin (bonding), thymosin (immune), acetylcholine (vagal tone)
Energetic Function
Love, compassion, empathy, connection, emotional balance, bridge between lower/upper chakras
3. Manipura (Solar Plexus)
Upper abdomen
Anatomical Correlate
  • Celiac plexus (solar plexus)
  • Pancreas (blood sugar regulation)
  • Adrenal glands (stress response)
  • Stomach, liver, gallbladder
Physiological Function
Digestion, glucose metabolism, stress hormone production, energy regulation
Hormones
Insulin, glucagon, cortisol, adrenaline (epinephrine), norepinephrine
Energetic Function
Personal power, willpower, confidence, action, transformation (digestion as metaphor)
2. Svadhisthana (Sacral)
Lower abdomen (below navel)
Anatomical Correlate
  • Pelvic plexus
  • Gonads (ovaries/testes)
  • Reproductive organs
  • Bladder, lower intestines
Physiological Function
Sexual function, reproduction, fluid balance, elimination, creative drive
Hormones
Estrogen, progesterone, testosterone, oxytocin (during arousal/orgasm)
Energetic Function
Creativity, sexuality, pleasure, emotions, flow, sensuality, desire
1. Muladhara (Root)
Base of spine / perineum
Anatomical Correlate
  • Sacral plexus
  • Adrenal glands (fight/flight)
  • Base of spine, legs, feet
  • Colon, rectum
Physiological Function
Survival response, fight/flight activation, grounding, structural support, elimination
Hormones/Neurotransmitters
Cortisol, adrenaline, norepinephrine (acute stress response)
Energetic Function
Safety, security, survival, grounding, connection to earth/physical reality, tribal belonging
